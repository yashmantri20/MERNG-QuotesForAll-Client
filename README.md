# Quotes For All

Quotes for all is a web app similar to social media where a user can interact with other people.

## Features

* Create Post
* Follow other users
* Like and Comment on other's posts
* User Management.
* Caching using Apollo Client.

## Tech Stack

* ReactJS, Semantic-UI, GraphQL, MongoDB
